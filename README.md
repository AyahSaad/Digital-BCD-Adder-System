# 2-Digit BCD Adder (Structural Verilog)

This project implements a **2-digit BCD adder (8-bit)** using structural Verilog. The design is built from basic logic gates and arithmetic modules to perform decimal addition using Binary-Coded Decimal (BCD) representation.

## Course Information

This project was developed as part of the **Advanced Digital Systems Design** course in the **Electrical & Computer Engineering Department at Birzeit University**.

## Description

- Each decimal digit is represented using **4-bit BCD**.
- The system adds **two 2-digit BCD numbers**.
- Includes **Full Adders, Ripple Carry Adder, and BCD correction logic**.
- A **testbench** was implemented for functional verification through simulation.
  
## Design Components

* Basic logic gates with propagation delays
* **1-bit Full Adder**
* **4-bit Ripple Carry Adder**
* **BCD Correction Logic**
* Structural integration of modules to build the complete **2-digit BCD adder**

## Verification

A **testbench** was implemented to verify the functionality by testing different input combinations and validating the output results through simulation.

## Technologies

* Verilog HDL
* Digital Logic Design
* Hardware Simulation
